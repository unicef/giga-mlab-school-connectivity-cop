# IQB-Edu: the Moldova use case

- [IQB-Edu_Moldova_schools.ipynb](./IQB-Edu_Moldova_schools.ipynb): Notebook that loads M-Lab / Giga Meter measurements for schools in Moldova, calculates IQB scores, and presents main findings
- [pyproject.toml](./pyproject.toml): Libraries and dependencies needed to run the notebook 
- [iqbedu/](./iqbedu): Folder containing IQB-Edu code (wrapper code for IQB)
- [data/](./data): Files with M-Lab measurements and school metadata
